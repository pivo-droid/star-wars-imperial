# Маленький проект Django связанный с Звёздными Войнами
## Планировка проекта

- `manage.py` — Оболочка управления Django
- `imperial_db` — Пакет проекта Django
- `core` — Основное приложение, содержащее модели, представления, шаблоны и пользовательские команды управления


## Требования

- Python 3.8+
- Django (нужна версия, совместимая с проектом — установить через pip)

## Настройка

1. Создайте и активируйте виртуальную среду

2. Установите Django если он ещё не установлен

3. Примените миграции и создайте базу данных:

```powershell
python manage.py migrate
python manage.py populate_curated
```

4. (Необязательно) Создайте суперпользователя для доступа к административной панели:

```powershell
python manage.py createsuperuser
```

## Запуск сервера

```powershell
python manage.py runserver
```

Откройте http://127.0.0.1:8000/ в вашем браузере
